Clarity by Example (WIP): Hello World

Our first smart-contract will print the classic “hello world” message. Here’s the full source code.

 

(print "hello world")

To run the smart-contract, put the code in hello-world.clar and pipe it into the clarity-repl.

 

$ cat hello-world.clar | clarity-repl
clarity-repl v0.21.0
Enter "::help" for usage hints.
Connected to a transient in-memory database.
 
Events emitted {
  contract_event: {
    contract_identifier: "ST000.contract-2",
    topic: "print",
    value: '"hello world"'
  },
  type: "contract_event"
}
"hello world"
